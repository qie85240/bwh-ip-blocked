很多人在这两个问题上绕了很久，本文就一次性说清楚。

## 先搞清楚：搬瓦工"换 IP"有几种路径

搬瓦工换 IP 这件事，本质上有两条路：

**路径一：迁移机房（免费）**

在 KiwiVM 后台选"Migrate to another DC"，搬到另一个数据中心，IP 地址自动跟着换——这就是所谓"通过换机房换 IP"。免费，次数不限，15 分钟内完成，系统完成后发邮件通知。

有一个细节：如果迁去 B 机房再迁回 A 机房，大概率还是原来那个 IP，因为原 IP 可能还没被别人占用。

**路径二：付费换 IP（$8.79）**

不动机房，直接申请换一个新的 IP 地址。目前固定收费，具体以官方页面为准。付款后系统一般很快处理，等待时间通常在 30 分钟以内，最多不超过 24 小时。换完之后官方保证新 IP 全球可达，不在任何黑名单里。

## 核心问题：IP 被封了，能靠迁移机房来换 IP 吗？

直接说结论：**不能。**

这是很多人踩过的坑。IP 被封之后，搬瓦工后台会自动禁用"Migrate to another DC"功能。你点进去会看到一个错误提示：


Migration backend is currently not available for this VPS.
Please try again in 10-15 minutes. (734152)


有时候这个错误也会出现在服务器繁忙时，但如果你的 IP 已经被封，它**必然**会报这个错误——反复尝试也没用。

官方的逻辑是：切换机房是用来"优化线路"的，不是用来绕过 IP 封锁的。IP 封了，就得花钱换，没有免费路走。

所以，**搬瓦工不换机房换 IP** 的正确理解是：

- **IP 没被封时**：可以通过迁移机房来免费更换 IP 地址，这叫"迁机房顺带换 IP"
- **IP 已被封时**：迁机房的路被堵死，只剩付费换 IP 这一条路

## 怎么先确认自己的 IP 到底有没有被封？

不要光靠 ping——ping 不通不等于 IP 被封，也可能是本地网络问题、防火墙问题、服务没启动。

正确的检测方式：

1. 登录搬瓦工账号，进入 KiwiVM 控制面板
2. 访问 IP 检测页面（在 KiwiVM 里找到 blacklist check 功能）
3. 点击"Test Main IP"
4. 出现 **IP NOT BLOCKED** = 没封，出现 **IP BLOCKED** = 封了

出现 IP BLOCKED 之后，不用继续折腾重启或者重装系统，直接进入付费换 IP 的流程。

## 付费换 IP 完整操作流程

1. 登录搬瓦工账号（客户中心，不是 KiwiVM）
2. 进入 IP 更换申请页面（已登录状态下，在 Services 菜单里可以找到 IP Change 入口）
3. 从列表中选出**你想换 IP 的那台 VPS**——如果有多台，一定要认清主机名和当前 IP，选错了就麻烦了
4. 点击 "Request IP Change"，系统自动生成工单
5. 回到账户首页，付款（$8.79，支持支付宝、微信支付、PayPal、信用卡）
6. 等待处理。一般 30 分钟内完成；节假日可能稍慢，但不会超过 24 小时
7. 收到工单回复通知后，再去尝试新 IP

付完钱之后，原来的 IP 会被回收，新 IP 保证在国内可以正常访问。

## 换 IP 之前值得想清楚的一件事

$8.79 换一个 IP，还是直接买一台新的 VPS？

几乎所有情况下都是换 IP 更划算。重新买一台 VPS 不仅可能贵得多，还要重新配置环境、迁移数据，时间成本远不止这点钱。

有朋友刚买完没多久 IP 就出问题，想着反正用没多久，干脆退款。搬瓦工支持 30 天内无条件退款，这种情况可以先看看是申请退款合适还是付费换 IP 合适，取决于你在这台机器上已经做了多少配置。

## FAQ

**Q：搬瓦工可以不换机房只换 IP 吗？**

A：可以，走付费换 IP 的流程，$8.79 一次，不需要迁移机房，数据和配置全部保留。

**Q：IP 没被封，可以通过迁机房来换一个新 IP 吗？**

A：可以。IP 未被封时，"Migrate to another DC"功能正常使用，迁到新机房会分配新的 IP 地址。迁机房本身免费，次数不限，但每次迁移会消耗一定流量（取决于 VPS 存储大小）。

**Q：迁回原机房还是原来的 IP 吗？**

A：不一定。如果你原来的 IP 还没被别人占用，大概率还是原来那个；如果已经被分配出去了，就会给你一个新 IP。没有固定的时间规律。

**Q：IP 被封了不换 IP，等一等会自己恢复吗？**

A：官方说法是可能会，但时间从几天到几个月不等，不可预期。实际使用中，靠等来恢复的情况很少见。需要继续用的话，付费换 IP 更可靠。

**Q：搬瓦工免费换 IP 的政策还有吗？**

A：已经取消了。当初曾经有过每 70 天免费换一次的政策，后来彻底下线，目前只有付费换 IP 这一条路。
